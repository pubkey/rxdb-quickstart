p2p todos

This is a local first todo app that stores data locally with RxDB and replicates it P2P with WebRTC to other devices without sending the data throught any central server. Open this url in another browser/device/tab to test the replication:

The whole app is implemented without a framework in about 200 lines of TypeScript code. To learn more about how it works, I recommend looking a the source code and read the RxDB Quickstart Guide.
